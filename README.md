# 溢彩包装 Production OS

> EVA 片材加工厂 MES 生产控制系统 — 材料约束驱动 + 工序流转 + 产线调度 + 追溯反馈

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + TypeScript + Vite + Pinia |
| 后端 | Express + better-sqlite3 |
| 数据库 | SQLite (WAL 模式) |
| 认证 | JWT + bcrypt |

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发环境
pnpm dev
```

访问 http://localhost:5173
