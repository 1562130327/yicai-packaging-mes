## 状态流转

### Task 状态机

```
                    ┌─────────────┐
                    │   pending   │
                    └──────┬──────┘
                           │ assign
                           ▼
                    ┌─────────────┐
                    │  assigned   │
                    └──────┬──────┘
                           │ start
                           ▼
                    ┌─────────────┐
              ┌────→│  running    │←────┐
              │     └──────┬──────┘     │
              │            │            │
              │     complete│     pause  │
              │            ▼            │
              │     ┌─────────────┐     │
              │     │  completed  │     │
              │     └─────────────┘     │
              │                         │
              └─────────────────────────┘
                    (重新开始)

    任何时候都可以: cancel → cancelled
```

### WorkflowStep 状态机

```
    waiting → ready → running → done
                        ↓
                    paused (异常暂停)
```
